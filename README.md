# Vivek K K - Cinematic Portfolio

A premium, cinematic portfolio website built with React, TypeScript, Tailwind CSS, Framer Motion, GSAP, and Lenis.

The design uses a dark crimson red and black visual identity with smooth scroll storytelling, animated section transitions, interactive project panels, timeline reveals, animated counters, and responsive typography.

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Framer Motion
- GSAP + ScrollTrigger
- Lenis
- Vite

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```

## Build

Create a production build:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Project Structure

```text
src/
  App.tsx       Main portfolio sections and animation logic
  main.tsx      React app entry point
  styles.css    Tailwind layers and custom cinematic styling
```

## Customization

Update contact links, resume path, project links, and profile URLs in `src/App.tsx`.

Current placeholders include:

- `vivekkk@example.com`
- LinkedIn URL
- GitHub URL
- Phone number
- `/resume.pdf`

Place a resume file in `public/resume.pdf` or update the resume link to your hosted file.
